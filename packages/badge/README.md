# `@posten-hedwig/badge`

Hedwig Badge provides various style badges

## Styles

- Badge
- DarkBadge
- WarningBadge
- WhiteBadge

## Sizes

- smaller
- small
- large
- larger

## Usage

```bash
npm install @posten-hedwig/badge
```
