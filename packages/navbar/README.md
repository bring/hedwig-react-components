## Usage

### Install

Install Navbar

```sh
npm install @posten-hedwig/navbar
```

### Import

Import the Navbar component

```js
import { Navbar } from '@posten-hedwig/navbar'
```

### Render

Render the Navbar

```jsx
<Navbar
    logo='posten'
    logoAriaLabel='Go to homepage'
    logoHref='#'
    logoTitle='Posten Logo'
    skipToMainHref='#content'
    skipToMainTitle='Main content' />
```
